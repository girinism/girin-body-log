# GIRIN BODY LOG

> AI 빅웨이브를 넘기 위한 신체+정신 컨디션 트래킹 시스템. 혼자가 아니라 길드와 함께.

## 왜 만들었는가

폐관수련 8주 동안 AI와 코드만 파면 몸이 망가진다.
신체가 건강해야 정신이 버티고, 정신이 버텨야 AI가 만드는 BIG WAVE를 탈 수 있다.
혼자 하면 3일 만에 포기한다 — 길드원과 함께 봐야 지속된다.

## 사용자

| 역할 | 이름 | 기능 |
|------|------|------|
| 대표 | 박승현 (기린) | 운동 기록, 신체 수치, 대시보드 |
| 길드원 | 김민호, 이승훈 | 각자 기록 입력, 서로 현황 확인 |
| PT 트레이너 | 추후 섭외 | 전체 멤버 조회, 피드백 |

## 기록 항목

- 운동 기록 (종류, 횟수, 시간)
- 신체 수치 (체중, 체지방률, 컨디션 1~10)
- 수면 시간
- 일일 루틴 체크 (아침/브레이크/저녁)

## 기술 스택

- HTML + CSS + JavaScript
- Firebase (데이터 저장)
- Vercel 또는 GitHub Pages (배포)
